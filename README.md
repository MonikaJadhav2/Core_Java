# Core Java

This repository documents my journey of learning Core Java from fundamentals.
It contains concept-wise programs written while understanding how Java works internally.

This repo is created for learning, practice, and revision purposes.

---

## Concepts Covered

### 1. Java Basics
- JVM, JRE, JDK
- Data types
- Variables and scope
- Type casting

### 2. Object-Oriented Programming (OOP)
- Class and Object
- Constructor
- Inheritance
- Polymorphism
- Abstraction
- Encapsulation

### 3. Keywords in Java
- static
- final
- this
- super

### 4. String Handling
- String Constant Pool
- == vs equals()
- intern() method
- String immutability
- String methods

### 5. Exception Handling
- try-catch
- finally
- throw vs throws
- custom exceptions

### 6. Multithreading
- Thread class
- Runnable interface
- Thread lifecycle
- Synchronization (basics)
